GET /api/apple/devices.json

Returns a comprehensive historical list of Apple devices, including Macintoshes, iPhones, iPads, and accessories. Features metadata about release years, manufacturing budgets, and tech specifications. Retorna uma lista histórica abrangente de dispositivos Apple, incluindo Macintoshes, iPhones, iPads e acessórios. Apresenta metadados sobre anos de lançamento, orçamentos de fabricação e especificações técnicas.

Data Model (Response) Modelo de Dados (Resposta)

Property Type Description
id string Unique identifier for the device (slug).
name string Official name of the Apple device.
category string General category classification.
photo string (URL) Direct URL to a 300px thumbnail on Wikimedia Commons.
release_year number | string Year the device was publicly released.
development_year number | string Estimated year the internal project/R&D started.
manufacturing_budget string Historical estimated or exact project cost.
specs object Contains keys like Processor, Memory, Storage.

Example Usage (JavaScript) Exemplo de Uso (JavaScript) 

// Import our built-in ApiRouter
import { ApiRouter } from '/js/api-router.js';

// Initialize router for the apple category
const router = new ApiRouter('apple');

async function fetchAppleDevices() {
    try {
        // Fetch the JSON payload
        const data = await router.fetchResource('devices');
        
        console.log(`Loaded ${data.length} Apple devices!`);
        console.log(data[0].name); // e.g. "Apple II"
        
    } catch (error) {
        console.error("Failed to load API data:", error);
    }
}

fetchAppleDevices();
